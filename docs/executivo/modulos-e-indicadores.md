# Modulos e Indicadores

## Modulos do sistema

| Modulo | Objetivo institucional | Perfis principais |
|---|---|---|
| Dashboard | visao rapida da operacao escolar | gestao |
| Alunos | cadastro e manutencao academica | coordenacao, secretaria |
| Professores | cadastro e organizacao docente | coordenacao, administracao |
| Turmas | estrutura academica e distribuicao | coordenacao |
| Disciplinas | curriculo e componentes | coordenacao |
| Notas | registro de desempenho | professor, aluno, gestao |
| Frequencia | controle de presenca | professor, aluno, gestao |
| Atividades | tarefas, entregas e avaliacao | professor, aluno |
| Calendarios | agenda institucional e docente | todos ou perfis pedagogicos |
| Comunicados | comunicacao segmentada | gestao, professores, alunos |
| Biblioteca | acervo e emprestimos | secretaria, alunos, professores |
| Relatorios | visao gerencial e pedagogica | gestao |
| Diario | registro de aula e planejamento | professor |
| Metas | acompanhamento pessoal do aluno | aluno |
| Usuarios | acesso, perfis e credenciais | administracao, coordenacao |

## Indicadores que o sistema pode suportar

- total de alunos ativos
- total de professores ativos
- total de turmas em andamento
- distribuicao de alunos por turno
- presenca por periodo
- media por disciplina
- atividades pendentes e entregues
- eventos institucionais futuros
- emprestimos ativos da biblioteca
- perfis pendentes de aprovacao

## Uso gerencial recomendado

- usar o dashboard como ponto de acompanhamento diario;
- usar relatorios para leitura mensal de indicadores academicos;
- usar gestao de usuarios para governanca de acesso;
- usar o calendario escolar como referencia unica de agenda institucional.
